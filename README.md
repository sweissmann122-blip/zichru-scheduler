# Zichru Scheduler (Ready to Deploy)

This is a ready-to-deploy Vite + React + TypeScript + Tailwind project for your Zichru Scheduler.

## One-minute deploy (Vercel)

1) Go to https://vercel.com and sign up with your Google or GitHub account.
2) Click **Add New Project** → **Import** from GitHub.
3) If you don't have a repo yet: create one at https://github.com/new (any name, e.g., `zichru-scheduler`), then upload these files (drag-and-drop works on the repo page).
4) Back on Vercel, select the repo → **Framework Preset: Vite** → Keep defaults (Build Command: `vite build`, Output: `dist`), click **Deploy**.
5) Done! You'll get a public URL like `https://zichru-scheduler.vercel.app`.

## Google Sites (embed)
- Open Google Sites → **Insert** → **Embed** → paste your Vercel URL → **Insert**.

## Local dev (optional)
```bash
npm install
npm run dev
# open the URL it prints (usually http://localhost:5173)
```

## Build (optional)
```bash
npm run build
npm run preview
```
